[DBeaver 실습: DB 연결 및 테이블 직접 만들기]

DBeaver는 DB 내부를 눈으로 보고 직접 쿼리를 날려볼 수 있는 필수 도구입니다.

1단계: 프로젝트 DB 연결하기
- Host: localhost
- Port: 56662 (매번 바뀔 수 있으니 로그 확인 필수)
- Database: mydatabase
- Username: myuser
- Password: secret
- [Test Connection] 버튼으로 연결 성공 확인

2단계: 연결된 정보 확인
- Database Navigator에서 'mydatabase > schemas > public > tables' 경로로 이동
- `schema.sql`에 의해 이미 만들어진 `users`, `boards` 테이블이 있는지 확인

3단계: SQL 편집기 열기
- 상단 메뉴 [SQL 편집기] > [새 SQL 편집기] 선택 (단축키: Ctrl + Enter)

4단계: 연습용 테이블 직접 만들기 (SQL 실습)
- 아래 쿼리를 입력하고 실행(Ctrl + Enter 또는 주황색 화살표 클릭)
```sql
CREATE TABLE test_member (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100) UNIQUE
);
```

5단계: 데이터 직접 넣어보기
- 만들어진 `test_member` 테이블을 더블 클릭하고 [데이터] 탭에서 [행 추가(Alt+Insert)]를 눌러 직접 값을 입력 후 저장(Ctrl+S)해보기

[학습 계획: 유저 기반 게시판 만들기]

본 계획은 MyBatis 체제에서 전형적인 SI 스타일의 게시판을 직접 구현하며 실무 역량을 쌓는 과정입니다.

1단계: 기초 CRUD 구현 (기본기)
- 목표: 게시판의 가장 기본적인 Read, Create 기능을 완성합니다.
- 상세: 
  - DB `board` 테이블 설계 및 생성
  - 게시글 목록 조회 (List) 및 상세 조회 (Detail)
  - 게시글 등록 (Write) 처리
- 핵심 학습: Mapper XML 작성법, @Service-@Mapper 연동

2단계: 게시판 기능 고도화 (실무 환경)
- 목표: 실제 서비스에서 필수적인 기능들을 추가합니다.
- 상세:
  - 게시글 수정 (Update) 및 삭제 (Delete)
  - 조회수 증가 로직 (상세 조회 시 +1)
  - 페이징(Paging) 처리 (MyBatis의 동적 쿼리 활용)
  - 제목/내용 기반 검색 기능 추가
- 핵심 학습: XML 내 동적 쿼리(`<if>`, `<where>`), 페이징 쿼리

3단계: 유저 연동 및 보안 (Spring Security 활용)
- 목표: 로그인한 사용자만 글을 쓸 수 있게 보안을 강화합니다.
- 상세:
  - 게시글 작성 시 현재 로그인한 유저 ID 저장
  - 본인 글만 수정/삭제 가능하도록 권한 체크 로직 추가
  - 게시판 목록 등에 작성자 이름 표시
- 핵심 학습: SecurityContextHolder 사용법, 권한 부여(Role-based Authorization)

4단계: 부가 기능 및 마무리
- 목표: 게시판의 완성도를 높이는 부가 기능을 추가합니다.
- 상세:
  - 댓글(Comment) 기능 추가 (일대다 관계 실습)
  - 파일 업로드 및 다운로드 기능 (MultipartFile)
  - 게시판 이동 및 로그아웃 등 UI 편의성 개선
- 핵심 학습: 테이블 간 관계 처리, 파일 입출력 처리
